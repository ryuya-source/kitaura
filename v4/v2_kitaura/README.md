# 北浦レイクサイドRVパーク Webサイト

## 概要
Next.js 14 (App Router) + TypeScript + Tailwind CSSで構築したRVパークの公式Webサイトです。
UI仕様書（`v4/実装UI仕様/UI仕様書`）に準拠して実装されています。

## 技術スタック
- **フロントエンド**: Next.js 14 (App Router) + TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **アイコン**: Lucide React
- **フォーム**: FormSubmit
- **ホスティング**: Vercel
- **CMS**: microCMS（お知らせ管理）

## デザイン仕様の適合

### カラーパレット
- **背景色**: `#FAF8F5`（淡いベージュ）
- **カード背景**: `#FFFFFF`（白）
- **テキスト**: `#1C1C1C`（濃いグレー）、`#6B6B6B`（グレー）
- **ボーダー**: `#E8E5E0`（薄いグレー）
- **アクセント**: `#5A8A5A`（緑）、`#1E3A5F`（ネイビー）
- **警告**: `#FFF8E6`（薄い黄色）、`#8B6914`（ゴールド）
- **ヒント**: `#F0F7F0`（薄い緑）、`#3D6B4F`（深緑）

### タイポグラフィ
- **フォントファミリー**: Inter
- **ページタイトル**: 23px, font-light, tracking-[-0.5px]
- **セクションタイトル**: 12px, font-medium, tracking-[-0.3px]
- **本文**: 9px, font-normal
- **キャプション**: 7px, font-normal

### レイアウト
- **画面幅**: 最大375px（スマートフォン）
- **padding**: [32px, 10px]（上下32px、左右10px）
- **gap**: 32px（セクション間）
- **カード角丸**: 7px または 12px
- **ボーダー**: 1px solid

## セットアップ

### 必要な準備
1. Google Maps APIキー
2. Google Analytics 4 測定ID
3. microCMS APIキー
4. YouTube Data APIキー

### インストール

\`\`\`bash
# 依存関係のインストール
npm install

# 環境変数の設定（.env.localは既に作成済み）
\`\`\`

### 開発サーバーの起動

\`\`\`bash
npm run dev
\`\`\`

ブラウザで [http://localhost:3000](http://localhost:3000) を開きます。

**Cursor内でプレビューする方法:**

1. **コマンドパレット** (Cmd/Ctrl + Shift + P) を開く
2. `Simple Browser: Show` を入力して実行
3. URL欄に `http://localhost:3000` を入力

または、

1. サイドバーの「ポート」タブを開く
2. ポート `3000` の行にある「地球儀アイコン」をクリック
3. Cursor内のブラウザでプレビューが開きます

### ビルド

\`\`\`bash
npm run build
npm start
\`\`\`

## 実装メモ

### UI仕様書に準拠した実装
- すべての色、フォントサイズ、スペーシングは仕様書通りに実装
- レイアウトは`vertical`（縦積み）を基本に、gap、paddingを厳密に適用
- カードコンポーネントは角丸、ボーダー、背景色を仕様通りに実装
- アイコンはLucide Reactを使用し、サイズと色を仕様書に合わせて調整

### アコーディオン機能
- アクセスページの「お車での所要時間」セクションでアコーディオンを実装
- クリックで開閉、ChevronDownアイコンが回転するアニメーション付き

### レスポンシブデザイン
- 最大幅375pxを基準にスマートフォン最適化
- Tailwind CSSのユーティリティクラスを活用
