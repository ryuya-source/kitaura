# 北浦_サイト改修

## 目的

`要件定義` を踏襲し、UIデザイン（プロトタイプ）を Next.js（`v4/v2_kitaura/`）に実装しています。

## UI

- 場所: `v4/v2_kitaura/`（Next.js）
- 起点: `http://localhost:3000`

### 起動（macOS）

```bash
cd v4/v2_kitaura
npm ci
npm run dev
```

## バージョン管理

このディレクトリ配下は Git で管理できる想定です（`.gitignore` 同梱）。

## かんたん起動（ターミナル不要）

- `open-ui.command` をダブルクリックすると dev server を起動してブラウザで開きます
