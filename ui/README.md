# 北浦レイクサイドRVパーク（UIプロトタイプ）

`要件定義` の方針（体験のビジュアル化 / 予約導線の強化 / ルールの見せ方改善 / 周辺情報の充実）を踏まえ、添付イメージのトーン（ベージュ×余白×写真中心）で静的UIを実装したものです。

## ブラウザで開く

### 方法1（推奨）
- `ui/index.html` をダブルクリックしてブラウザで開く

### 方法2（ターミナルから）
`北浦_サイト改修` ディレクトリで次を実行します。

```bash
open ./index.html
```

## 構成

- `ui/index.html`：1ページ完結のUI
- `ui/styles.css`：デザイン（ベージュ基調、カード、余白、追従CTA）
- `ui/script.js`：メニュー/カルーセル/カレンダー（ダミー）/年表示
- `ui/assets/*.svg`：ローカルで完結するダミー画像

## メモ

- 予約・料金・文言はすべてデモ用です（実運用時に外部予約/フォームへ接続想定）。
- 画像は差し替え前提で、レイアウト検証に必要な分だけSVGで用意しています。
