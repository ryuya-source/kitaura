RULES

利用ルール（分かりやすく）

トラブルになりやすいポイントは、文章を短く・見出しを明確に。
“守る理由” が伝わる言い回しで、安心感をつくります。

ペット同伴のルール +
  • リード着用を基本とし、共有スペースでは必ず管理をお願いします。
  • 排泄物は速やかに処理し、周囲が気持ちよく過ごせる環境づくりにご協力ください。
  • 犬種に関する条件がある場合は、ここに簡潔に明記して誤解を減らします。
電源利用の注意 +
  • 同時使用の機器によってブレーカーが落ちる場合があります（季節家電の併用に注意）。
  • 必要に応じて延長コードをご持参ください。
静かな時間（夜間マナー） +
  • 夜間は小さめの声量で。スピーカー等の音量は控えめにお願いします。
  • 消灯後の車の出入りは、周囲への配慮をお願いします。
